Общая информация

TPControl - это графический редактор для создания графического интерфейса, с дальнейшим его портированием на IPad. В результате работы программы, создается XML-файл, который одновременно является и файлом проекта приложения TPControl, что разрешает его дальнейшее редактирование, и файлом для экспорта в приложение на IPad.

Начало работы с программой

Для начала работы с программой перейдите в раздел меню: File->New project, либо нажмите комбинацию клавиш Ctrl+Shift+N. Далее необходимо выбрать папку, в которой вы хотите создать проект. Затем введите имя проекта. Проект создан. Теперь можно приступать к его наполнению. Если же необходимо открыть существующий проект для редактирования, выберите File->Open project, либо нажмите комбинацию клавиш Ctrl+Shift+O.
