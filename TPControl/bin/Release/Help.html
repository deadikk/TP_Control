Общая информация

TPControl - это графический редактор для создания графического интерфейса, с дальнейшим его портированием на IPad. В результате работы программы, создается XML-файл, который одновременно является и файлом проекта приложения TPControl, что разрешает его дальнейшее редактирование, и файлом для экспорта в приложение на IPad.

Начало работы с программой

Для начала работы с программой перейдите в раздел меню: File-> New project, либо нажмите комбинацию клавиш Ctrl+Shift+N.
Далее необходимо выбрать папку, в которой вы хотите создать проект.
Затем введите имя проекта. Проект создан. Теперь можно приступать к его наполнению.
Если же необходимо открыть существующий проект для редактирования, выберите File-> Open project, либо нажмите комбинацию клавиш Ctrl+Shift+O.

Окно программы и элементы управления

Окно программы состоит из многостраничного окна для размещения на нем графических элементов, окна свойств с правой стороны экрана а также верхней панели кнопок быстрого доступа.
Основное окно для размещения элементов имеет неограниченное число страниц, перемещатсья по которым можно на его нижней панели управления.
Окно свойсв элемента разделено на вкладки, в которых сгруппированы свойства элемента.
На панели кнопок быстрого доступа размещены альтернатиыные кнопки для всех действий основного контекстного меню а также строка поиска элемента по имени.

Страница и ее свойства

Каждая страница приложения является отдельным объектом со своими свойствами:
-имя страницы;
-картинки заднего фона для включенного и выключенного состояния кнопок
-цвет фона страницы
Для изменения значения определенного свойства дважды кликните по его полю в столбце Value.
Для добавления новой страницы можно воспользоваться кнопкой быстрого добавления на нижней панели страницы, из контекстного меню в разделе Panel-App page или воспользоваться комбинацией клавиш Ctrl+N.

Кнопка и уровень

Основные 2 элемента, которые используются для создания графического интерфейса-это кнопка и уровень.
Для их добавления можно воспользоваться соответствующими пунктами меню Button.
Add AMXButton для добавления кнопки (или комбинация Ctrl+1)
Add AMXLevel для добавления уровня (или комбинация Ctrl+2)
Так же можно включить режим рисования кнопок.
В данном режиме для начала необходимо нарисовать прямоугольних а затем выбрать какой элемент будет создан.
Для редактирования размера и расположения элементов, можно воспользоваться непосредственно способом Drag&Drop, либо задать точные координаты и размер в таблице свойств.
Редактирование внешнего вида для кнопки разделено на 2 вкладки, для включенного и выключеного состояний.
Редактирование свойсв вкладки Programming не влият на внешний вид элемента, однако необходим для его программирования и дальнейшего использования.
Для корректного размещения и взаимного перекрытия кнопок, можно располагать их в слоях глубже, либо наоборот ближе к поверхности.
Для этого можно воспользоваться соответствующими кнопками меню Layout, кнопками быстрого меню, либо комбинациями клавиш Ctrl+↓,Ctrl+↑.

После выполнения всех необходимых действий, сохранить проект можно из меню File-> Save project либо из панели быстрого доступа, или исползуя комбинацию клавиш Ctrl+Shift+S

Дополнительно

Если есть необходимость в изменении размера рабочей области приложения, к примеру, если необходимо разработать приложение для другого устройства с другим разрешением экрана, то это можно сделать в разделе Edit->Preferences.
Также в данном разделе есть возможность включить автосохранение через определенный промежуток времени, включить сохранение положения кнопок быстрого доступа и включить отображение курсора в виде прицела.
